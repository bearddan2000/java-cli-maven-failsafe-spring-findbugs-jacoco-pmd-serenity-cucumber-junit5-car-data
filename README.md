# java-cli-maven-failsafe-spring-findbugs-jacoco-pmd-serenity-cucumber-junit5-car-data

## Description
A POC for maven app using junit5
serenity, and cucumber framework
 with findbugs,
pmd and failsafe plugin.

## Tech stack
- java
- maven
	- spring
  - junit5
  - findbugs
  - jacoco
  - pmd
  - failsafe
  - cucumber
	- serenity


## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
